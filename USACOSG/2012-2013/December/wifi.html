This problem is quite similar to the umbrella problem in 2011-2012 December contest. We can solve this problem using dynamic programming.

In my code, dp[i] stands for the minimum cost we need to spend to cover every cow up to cow[i] inclusive. for dp[i], we can try to add a base station covering cow i to cow k, k is less than i, and pick the one yielding the least cost. thus, dp[i] = min(dp[k] + costOfLength(posOf(i) - posOf(k + 1)).
